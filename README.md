# MATLAB Projects

This repository is a parent container for multiple MATLAB project submodules.

Each submodule represents a separate MATLAB-based project using different MATLAB tools and techniques.

## Included submodules

- `Business-Inteligence`
- `Cartoon-Filter`
- `Data-Analytics`
- `Differential-Equations`
- `Digital-Signal-Processing`

## About

All submodules are independent projects, each demonstrating different MATLAB workflows, data processing, visualization, and analysis methods.

This repository is organized as a collection of related MATLAB projects, making it easier to manage and browse them together.
